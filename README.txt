The code is a simple basic exemple :using only json and requests to scrap the data from USA AirBnb offers.

Saving the data in json file and DB at the same time
Scheduler function integrated.
Google maps location preview integrated.
NB the project is working only for the USA.


Way to upgrate the code:
Install django-cities
Install autocomplete
Clean the data using pandas

1) Open the project in IDE
2) Go to setting and change DB settings
3)Run the project.
4) Queries can be done by country or by city and country
